# Импорт данных 1С в desktop-прототип

## Что приложение читает автоматически

- Все папки `exports_*` в корне проекта, если в них есть `manifest.csv`
- `model_schema/*.txt` для восстановления полного состава полей и табличных частей
- Последнюю доступную выгрузку по каждому объекту

## Что уже подключено в UI

- `Catalog.Контрагенты`
- `Catalog.Номенклатура`
- `Document.ЗаказПокупателя`
- `Document.СчетНаОплату`
- `Document.РасходнаяНакладная`
- схемы полей и табличных частей из `model_schema`

## Где смотреть

- В приложении: `Продажи -> Данные 1С`
- Вкладки показывают:
  - реестр карточек
  - все поля выбранной записи
  - табличные части выбранной записи
  - отдельную вкладку со схемами объектов

## Ограничения текущего шага

- Не все ссылочные поля пока можно красиво расшифровать в человекочитаемый текст, потому что часть служебных справочников еще не выгружена
- Рабочие формы продаж пока используют свою операционную модель, а вкладка `Данные 1С` служит мостом для миграции и проверки полноты переноса
